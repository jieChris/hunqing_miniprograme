# 婚庆小程序

一款基于微信小程序开发的婚庆服务系统，采用红色主题设计，提供婚礼策划、婚纱摄影、婚宴酒店、婚礼用车、婚礼用品等分类服务。

├── pages/                    # 页面目录

## 功能模块

### 1. 首页（index）
- **Banner轮播图**：展示活动宣传图片
- **分类按钮模块**：婚礼策划、婚纱摄影、婚宴酒店、婚礼用车、婚礼用品
  - 点击分类按钮调用 `fucClick` 函数，跳转到商户列表页
- **猜你喜欢模块**：推荐商品套餐
  - 点击商品卡片调用 `goodDetail` 函数，跳转到团购详情页

### 2. 商户列表页（store-list）
- **排序功能**：支持按距离、人气、商品种类排序
- **商户列表展示**：显示商户图片、名称、评分、距离、人气等信息
- **点击商户**：调用 `storelick` 函数，跳转到商户详情页

### 3. 商户详情页（store-detail）
- **商户信息**：图片轮播、基本信息、联系方式
- **服务项目**：展示商户提供的服务列表
- **网友点评**：展示用户评论和建议

### 4. 团购详情页（group-detail）
- **团购信息**：价格、评分、销量等
- **适用商户**：展示可使用该团购的商户列表
- **团购详情**：详细说明
- **购买须知**：使用规则和注意事项
- **网友点评**：用户评价
- **立即抢购**：调用 `gobuy` 函数进行购买

## 设计特点

### 配色方案
- **主色调**：红色系（#E91E63），营造喜庆氛围
- **辅助色**：金色（#FFD700），提升质感
- **背景色**：暖白色（#FFF8F0），柔和舒适

### UI风格
- 圆角设计：统一使用 8-16rpx 圆角
- 卡片式布局：白色背景，红色边框，轻微阴影
- 渐变按钮：红色渐变背景，金色文字
- 响应式设计：适配不同屏幕尺寸

## 技术实现

### 核心技术
- **WXML**：页面结构
- **WXSS**：样式设计（支持CSS变量）
- **JavaScript**：页面逻辑和交互
- **微信小程序API**：路由、网络请求、位置服务等

### 主要API使用
- `wx.navigateTo`：页面跳转
- `wx.request`：网络请求（封装在 utils/api.js）
- `wx.getLocation`：获取用户位置（用于距离计算）
- `wx.makePhoneCall`：拨打电话
- `wx.openLocation`：打开地图

## 开发说明

### 配置API地址
在 `app.js` 中修改 `globalData.baseUrl` 为实际的API地址：

```javascript
globalData: {
  baseUrl: 'https://your-api-domain.com/api'
}
```

### 图片资源
所有图片资源放在 `images/` 目录下，按用途分类：
- `common/`：公共图片（logo、占位符等）
- `category/`：分类图标
- `banner/`：轮播图
- `products/`：商品图片
- `stores/`：商户图片
- `icons/`：小图标

详细说明请查看 `images/README.md`

### 数据格式
当前使用模拟数据，实际开发时需要：
1. 在 `utils/api.js` 中实现真实的API调用
2. 确保数据格式与代码中的数据结构一致
3. 处理错误情况和加载状态

## 注意事项

1. **网络请求**：需要在微信公众平台配置合法域名
2. **位置权限**：获取用户位置需要用户授权
3. **图片资源**：建议使用CDN或云存储，控制单张图片大小在200KB以内
4. **小程序包大小**：主包限制2MB，注意控制资源大小
